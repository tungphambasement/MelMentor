To run the website
- download nodejs which includes npm
- download mysql workbench community version
- create new connection, make a new schema
- in application.properties of backend, change the username, password for the connection, change schema name to your schema
- adjust the port (if you want)
- cd to frontend
- run: npm install
- run command: npm start
- cd to backend
- install dependencies
- run command: mvn spring-boot:run
- enjoy
